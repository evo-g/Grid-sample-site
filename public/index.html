A fun project that was orginally given to me as a front-end developer test and thought it would be a nice way to showcase some of my talents. Using HTML, CSS, and JavaScript.

Has a simple express server to serve the content. The task was to make this mockup site and did what I could to make it spot on. The JavaScript creates the images from an object with other image properties, and appends them with a container div to house the image and the hover elements. This task can be easily created with a library like React.js, but proved much more difficult with vanilla JS. Side navagation also expands. The end result had me enjoying the use of vanilla JS and more appreciation for what it can do.

I do not own the copyrights to any of the images, perhaps only my own. Logo of company was removed and placed my name instead. Discretion links do not work only meant to be a single page mockup. Was working for a mobile first approach. Thanks for checking it out have a nice day!
